# Spotify Clone 🎵

A responsive Spotify-inspired music player UI built using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Sidebar Navigation
- Music Cards
- Sticky Navbar
- Interactive Music Player
- Progress Bar
- Volume Control
- Hover Animations
- Dark Theme

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
